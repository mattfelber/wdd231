SITE PLAN

Site Name: Matt's Terminal Playground
Reasoning: Define the interactive nature of the website, referencing a terminal environment, and my name.
Site Purpose: This website offers a simulated terminal experience for users to explore basic commands and functionalities, introduction to my work through the world of command-line interface.
Scenarios:
Who is this website for? (Target Audience: Beginners interested in learning web development, businesses, professionals.)
The website caters to individuals to explore basic commands and find out more about my experience and background.
What commands can I use? (Target Audience: Users exploring the terminal)
The website offers a variety of fundamental commands displayed in a clear and concise manner. Users can experiment with these commands within the simulated terminal to understand their functionalities.
Color Schema:
Background Color: #000 (Black) - Creates a classic terminal look, enhancing readability of white text and code elements.
Text Color: #FFF (White) - Provides clear and sharp text visibility against the black background.
Typography:
Font: "Source Code Pro" (Monospace) - This font family is specifically designed for code readability, with a command line feel. With fixed-width characters and clear letter spacing, ideal for displaying commands and output in the simulated terminal.
Used for: All text within the simulated terminal (headings, commands, output)
Wireframe: linux terminal, minimal, black and matrix green, “Type --help for commands”
Mobile View: “Type --help for commands” (smaller terminal, no big ASCII name on top)
Top Section (20%): Site logo (text or image) centered.
Main Section (70%): Simulated terminal with clear borders, displaying a welcome message and blinking cursor.
Bottom Section (10%): Simple copyright information or developer name.
Desktop View:
Top Section (15%): Site logo (text or image) centered, with optional navigation bar for future expansion.
Main Section (70%): Simulated terminal occupying a larger portion of the screen for enhanced user experience.
Bottom Section (15%): Copyright information, developer name, and potential links to social media or contact details.
